## Mobilní aplikace pro zobrazení aktuálního počasí, 5denní předpovědi a kvality ovzduší (AQI) s vyhledáváním měst, oblíbenými položkami a nastavením měrných jednotek.

 ## Funkce
 
- Current Weather: aktuální počasí, podrobnosti (vlhkost/vítr/tlak), AQI
- 5denní předpověď
- Vyhledávání: vyhledávání města
- Oblíbené: seznam oblíbených měst, přecházení mezi městy na hlavní obrazovce
- Nastavení: měrné jednotky (°C/°F), téma
- Moje poloha: dotaz na počasí podle aktuální geolokace

## Technologie

- Kotlin
- Jetpack Compose
- MVVM (ViewModel + UI state)
- Navigation (Compose)
- Retrofit (REST)
- DataStore (persistent storage)
- Fused Location Provider (geolocation)

##  API klíč (povinné)
Aplikace potřebuje klíč OpenWeather
Klíč vložte do pole value v  build.gradle
